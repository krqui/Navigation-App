# Navigation-App

The purpose of this project is to learn about and practice the benefits of React Navigation in React Native.
This app is optimized for both Android and IOS devices.

## Features of Stack Navigator
1. **Supports navigation between screens using a stack-based approach.**
2. **Provides a header bar at the top of the screen that displays the current screen title and back button.**
3. **Allows customization of the header bar with custom titles, buttons, and styles.**
4. **Supports animations when transitioning between screens.**
5. **Supports nested stacks and tabs for more complex navigation structures.**

## React Native and React Navigation skills gained during this project.
- Stack Navigator
- Native Stack
- Drawer Navigator (createDrawerNavigator, DrawerContentComponentProps,DrawerContentScrollView)
- Bottom Tabs (BottomNavigation)
- Material Bottom Tab Navigator
- Material Top Tab Navigator
- Ionicons icon set implemented.
        
- Context and global state of the app implemented.
- Trigger actions from different screens.

- React Native (useWindowDimensions, View, Image, Text)
- React-native-gesture-handler
